# 版本路线图

最后更新：2026-05-14

## 总体策略

路线从“首页重图片转换工具”调整为：

```text
轻首页
-> 核心编辑器
-> 图片导入转换
-> 后台 CMS
-> 原创图纸库
-> 分类页和教程页 SEO
-> AdSense
-> 用户保存和分享
```

当前已有图片转换原型是有价值的，但它应该迁移到 `/convert` 和 `/editor`，不应该继续堆在首页。

## V0.1A 当前原型整理

目标：把已有功能从首页重工具中拆出来，形成清晰路由。

范围：

- 首页改轻。
- 新建 `/editor`。
- 新建 `/convert`。
- 保留已有图片导入、色卡匹配、颜色限制、预览、用豆统计、PNG 导出能力。
- 将共用逻辑拆到 `src/lib/pattern`。
- 将 UI 拆到 `components/editor` 和 `components/convert`。

不做：

- 登录。
- 用户云端保存。
- 社区投稿。
- AdSense 广告位。

验收标准：

- `/` 可以清晰分流到编辑器、转换器和图纸库。
- `/editor` 可以从零创建图纸。
- `/convert` 可以把图片生成草稿并进入编辑器。
- 首页不再承担完整工具交互。

## V0.1B 编辑器基础版

目标：让用户真正能自定义拼豆图纸。

范围：

- 画布尺寸设置。
- 网格编辑。
- 画笔。
- 橡皮。
- 填充。
- 色卡选择。
- 格子色号显示。
- 坐标编号。
- 颜色统计。
- PNG 导出。

验收标准：

- 用户不用上传图片，也能从空白画布做图纸。
- 用户能修改图片转换生成的草稿。
- 导出的图纸能用于实际制作。

## V0.2A 后台 CMS 基础版

目标：建立自用后台，让图纸库可以持续发布，而不是靠修改静态数据维护。

范围：

- PostgreSQL 和 Prisma。
- AdminUser、Category、Pattern、Guide 数据模型。
- `/admin/login` 管理员登录。
- `/admin/patterns` 图纸列表。
- `/admin/patterns/new` 和 `/admin/patterns/[id]`。
- 图纸草稿、发布、下架状态。
- 图纸 SEO 字段：title、description、summary、seoTitle、seoDescription。
- 管理员在后台手动创建图纸草稿、导入 Pattern JSON、上传预览图。

验收标准：

- 管理员可以登录后台。
- 管理员可以创建图纸草稿并发布。
- 未发布内容不会出现在公开页面和 sitemap。
- 后台页面不被搜索引擎索引。

## V0.2B 内容和 SEO 基础版

目标：建立图纸库和 SEO 页面矩阵。

范围：

- `/patterns` 图纸库。
- `/categories/[categorySlug]` 分类页。
- `/pattern/[patternSlug]` 图纸详情页。
- `/guides` 教程列表。
- `/guides/[guideSlug]` 教程详情页。
- 第一批 20-30 个通过后台发布的原创 beginner patterns。
- 3-5 篇教程文章。
- sitemap、robots、基础结构化数据。
- GA4 和 Search Console。

验收标准：

- Google 能抓取主要公开页面。
- 每个图纸详情页有独特内容。
- 分类页不是空列表。
- 教程页能自然链接到编辑器和转换器。

## V0.3 导出和打印增强版

目标：让图纸更接近真实制作图纸。

范围：

- PNG 导出增强。
- PDF 导出。
- 每格色号导出。
- 坐标编号导出。
- 底部颜色清单。
- 打印尺寸适配 A4 / Letter。

验收标准：

- 导出图纸包含坐标、色号、颜色统计。
- 用户可以照着导出文件制作。
- 大尺寸图纸有分页或缩放策略。

## V0.4 AdSense 准备版

目标：在内容质量足够后申请 AdSense。

范围：

- ads.txt。
- Privacy Policy、Terms、Contact、Copyright 页面。
- 广告组件。
- 广告位只放在内容页和安全位置。
- 工具按钮、上传区、下载区附近不放广告。
- 未审核 UGC 不展示广告。

验收标准：

- 站点内容足够，不是单页工具。
- 广告不会干扰编辑器和转换器。
- 隐私政策覆盖 cookies、Analytics、AdSense。

## V0.5 用户保存与分享

触发条件：

- 自然搜索流量稳定。
- 编辑器和转换器有持续使用。
- 用户明确表达保存、分享、投稿需求。

范围：

- 登录。
- 私有保存图纸。
- Share to gallery。
- 审核队列。
- 举报入口。
- 公开图纸页审核后索引和展示广告。

不做：

- 评论。
- 私信。
- 关注流。
- 复杂积分系统。

## V1.0 社区和商业化

范围：

- 用户个人页。
- 收藏。
- 点赞。
- 公开 Gallery。
- 设计师投稿。
- 更强搜索。
- 耗材 affiliate。
- 高级 PDF 模板。
- 教师活动包。
- 批量转换。
