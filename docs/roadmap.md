# 版本路线图

## 总体策略

先做能被用户和搜索引擎理解的最小闭环：

```text
工具可用
-> 原创图纸库可浏览
-> 页面可收录
-> 合规页面完整
-> 申请 AdSense
-> 有自然流量后再做用户分享和社区
```

不要在没有流量前过早建设社区。社区功能只有在有人持续访问、保存、投稿时才会产生价值。

## V0.1 快速上线版

目标：尽快让英文站有一个可用工具和基础内容，可以开始被 Google 收录。

建议周期：3-7 天。

功能范围：

- 英文首页，首屏就是图片转图纸工具。
- 浏览器本地图像转换，不上传原图。
- 基础设置：尺寸、颜色数量、色卡选择、预览模式。
- 导出 PNG。
- 10-20 个原创 beginner patterns。
- 基础分类页。
- `privacy-policy`、`terms`、`contact`、`copyright`。
- sitemap、robots.txt、基础结构化数据。
- GA4 和 Search Console。

不做：

- 登录。
- 用户分享。
- 评论、点赞、收藏。
- 后台复杂审核。
- 在线高级编辑器。
- 服务器端图片处理。

验收标准：

- 用户可以上传图片并成功生成图纸。
- 移动端可用。
- Google 能抓取主要页面。
- 页面内容不是空壳，有实际图纸和说明。

## V0.2 内容与导出版

目标：让站点具备申请 AdSense 的内容厚度。

建议周期：1-2 周。

功能范围：

- 图纸库扩展到 50-80 个原创图纸。
- 图纸详情页包含尺寸、难度、颜色数、bead count、制作提示。
- 导出 PDF。
- 导出颜色清单。
- 图纸详情页相关内容推荐。
- 3-5 篇 guide 文章：
  - how to turn a photo into a bead pattern
  - beginner pin bead tips
  - bead color chart
  - how to choose bead pattern size
  - printable bead pattern guide
- 基础后台或脚本，用于录入原创图纸。

不做：

- 用户公开图库。
- 复杂搜索。
- 多人协作后台。
- 付费系统。

验收标准：

- 每个公开图纸详情页都有独特内容。
- 工具页、分类页、详情页内链完整。
- 站点可提交 AdSense 审核。

## V0.3 AdSense 准备版

目标：合规接入广告，不破坏工具体验。

建议周期：3-7 天。

功能范围：

- ads.txt。
- AdSense 代码配置。
- 广告位组件。
- Cookie/privacy messaging 预留。
- 广告只出现在内容页和安全位置。
- 工具按钮、上传区、下载区附近不放误导性广告。
- UGC 未上线或未审核页面不展示广告。
- 站点性能优化，避免广告造成严重布局偏移。

推荐广告页：

- 图纸详情页。
- 分类页。
- 教程文章页。

谨慎广告页：

- 图片生成工具页。
- 下载按钮附近。
- 移动端首屏。

验收标准：

- 隐私政策覆盖广告 cookies 和第三方广告说明。
- 页面布局不会诱导误点。
- 广告加载不会压垮 Core Web Vitals。

## V0.4 用户保存与分享版

触发条件：自然流量稳定后再做。

建议触发参考：

- 月自然搜索访问 1,000+。
- 工具页有持续使用记录。
- 有用户通过邮件或社媒表达保存、分享、投稿需求。

功能范围：

- Auth.js 登录。
- 用户私有保存图纸。
- Share to gallery。
- 投稿授权确认。
- 审核队列。
- 管理后台 approve / reject。
- 举报入口。
- 审核通过后才生成公开页和展示广告。

不建议同时做：

- 评论。
- 关注流。
- 站内消息。
- 复杂积分。

验收标准：

- 未审核内容不会被索引。
- 未审核内容不展示广告。
- 用户能删除自己的投稿。
- 管理员能快速下架违规图纸。

## V1.0 社区与规模化版

触发条件：内容库和工具已有稳定流量，用户投稿有明显需求。

功能范围：

- 公开 Gallery。
- 用户个人页。
- 收藏。
- 点赞。
- 标签筛选。
- 更强搜索。
- 设计师投稿入口。
- 批量生成 Open Graph 图片。
- AI/规则辅助审核。
- 邮件通知。

商业化扩展：

- 拼豆耗材 affiliate。
- 高级 PDF 模板。
- 无水印导出。
- Teacher activity pack。
- 批量转换。
- 付费图纸包。

## 流量起来前可以延后的功能清单

优先延后：

- 社区动态流。
- 评论区。
- 用户互相关注。
- 站内私信。
- 排行榜。
- 付费会员。
- 原生移动 App。
- 多语言社区。
- 复杂 AI 审核。
- 全自动背景移除。
- 云端保存所有上传原图。

优先保留：

- 工具体验。
- 页面速度。
- 原创图纸质量。
- 下载体验。
- 合规页面。
- SEO 标题和内链。
- 可维护的数据结构。
