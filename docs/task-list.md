## 当前产品结论

Pinbead 的方向调整为：

```text
拼豆图纸编辑器 + 可打印拼豆图库 + 图片导入转换工具
```

图片转拼豆图只生成草稿，不再作为最终成品。真正核心是 `/editor`，用户可以从零绘制，也可以导入图片草稿后继续修眼睛、轮廓、颜色和细节。

推荐 URL：

```text
/                              首页
/editor                        拼豆图纸编辑器
/convert                       图片转拼豆图纸
/patterns                      免费图纸库
/categories/[categorySlug]     分类页
/pattern/[patternSlug]         单个图纸详情页
/guides                        教程列表
/guides/[guideSlug]            教程详情页
```

## 3. 编辑器核心功能

### [x] 3.1 实现可编辑网格画布

需求：

- 使用可交互网格展示图纸。
- 支持点击单个格子上色。
- 支持拖拽连续绘制。
- 支持空白格。

验收标准：

- 用户可以在空白画布上绘制图案。
- 网格在桌面和移动端都不溢出。
- 大小变化不会导致布局跳动。

### [x] 3.2 支持画笔、橡皮和填充

需求：

- 画笔：用当前选中颜色绘制。
- 橡皮：清空格子。
- 填充：填充相邻同色区域。
- 工具选择要清晰，适合触屏和鼠标操作。

验收标准：

- 三种基础工具可用。
- 填充不会卡死浏览器。
- 当前工具状态在 UI 上清楚可见。

### [x] 3.3 支持坐标编号和色号显示

需求：

- 顶部、左侧、右侧、底部显示坐标编号。
- 每个格子可显示颜色 code，例如 H2、E8、PB-01。
- 提供开关：显示色号 / 隐藏色号。
- 字号要适合不同 grid 尺寸，避免文字溢出。

验收标准：

- 生成的图纸接近用户上传参考图那种可制作样式。
- 坐标和色号在导出中也可复用。

### [x] 3.4 支持色卡和颜色统计

需求：

- 色卡面板显示颜色名称、色号、HEX。
- 用户可以选择当前画笔颜色。
- 颜色统计实时更新。
- 统计包括颜色、色号、数量、总用豆数。

验收标准：

- 编辑任意格子后统计同步更新。
- 统计数量总和等于已填充格子数。

### [x] 3.5 支持导出 PNG

需求：

- 导出当前编辑器图纸。
- 支持网格线、坐标编号、格子色号、底部颜色清单。
- 文件名包含 pinbead、尺寸和时间戳。

验收标准：

- 导出的 PNG 可直接用于制作参考。
- 导出内容与编辑器当前状态一致。
