# Pinbead Docs

这个目录用于沉淀 Pinbead 站点的产品、技术、内容、合规和版本规划。

当前定位：面向海外英文市场的拼豆图纸工具站，核心是将用户上传的图片在浏览器本地转换为可打印的 pin bead / fuse bead / pixel bead pattern，同时用原创图纸库承接 SEO 与 AdSense 收益。

## 核心原则

1. 私人图片生成尽量在浏览器本地完成，不默认上传服务器。
2. 公开图纸库优先使用原创或授权安全内容，不依赖未经审核的用户上传图片。
3. 先做可被搜索引擎理解的高质量内容，再申请和放大 AdSense。
4. 网站流量没起来前，社区、登录、评论、复杂编辑器和付费系统都可以后置。
5. 使用自有服务器部署应用，但前期不必把所有基础设施都自建。
